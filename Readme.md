# A-Maze-Ing

### Descricao


O Makefile vai criar uma venv e baixar todas as dependencias necessarias para dentro dela. Para rodar O a-maze-ing de forma simples e rapida.

Uma venv (ambiente virtual) é um espaço isolado onde podes instalar e gerir dependências de um projeto sem afetar o sistema principal. Isto é especialmente útil quando precisas de uma versão específica de uma biblioteca que é incompatível com a que tens instalada no computador. Ao terminares o projeto, basta eliminares a pasta da venv e tudo o que foi instalado desaparece, deixando o teu sistema limpo e intacto.
