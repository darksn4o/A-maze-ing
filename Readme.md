# A-Maze-Ing

### Descricao


O Makefile vai criar uma venv e baixar todas as dependencias necessarias para dentro dela. Para rodar O a-maze-ing de forma simples e rapida.

Uma venv (ambiente virtual) é um espaço isolado onde podes instalar e gerir dependências de um projeto sem afetar o sistema principal. Isto é especialmente útil quando precisas de uma versão específica de uma biblioteca que é incompatível com a que tens instalada no computador. Ao terminares o projeto, basta eliminares a pasta da venv e tudo o que foi instalado desaparece, deixando o teu sistema limpo e intacto.

# Extra


1. Clonar o projeto do colega (primeira vez)
```bash
git clone https://github.com/utilizador/projeto.git
```
2. Criar o teu branch e mudar para ele
```bash
git checkout -b meu-branch
```
3. Ver em que branch estás
```bash
git branch
```
4. Depois de fazeres alterações, guardá-las
```bash
git add .
git commit -m "descrição do que fizeste"
```
5. Enviar o teu branch para o repositório remoto
```bash
git push origin meu-branch
```
6. Buscar as atualizações do colega (sem juntar ainda)
```bash
git fetch
```
7. Atualizar o teu main local
```bash
git checkout main
git pull origin main
```
8. Trazer as atualizações do main para o teu branch
```bash
git checkout meu-branch
git merge main
```
